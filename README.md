# 自动填充九宫格

## 使用方法

1. git clone 或 下载代码
2. 确定本地已经安装python3
3. cd到文件目录并执行python Sudoku.py
4. 逐行输入待填充的九宫格内容，需要自动填充的写0
```
Input line #1:
0,0,0,0,0,0,0,0,6
Input line #2:
0,0,7,0,9,0,5,0,3
Input line #3:
0,3,0,4,0,0,0,0,9
Input line #4:
4,8,3,0,0,6,0,2,0
Input line #5:
0,0,1,0,4,0,6,0,0
Input line #6:
0,6,0,3,0,0,9,4,8
Input line #7:
8,0,0,0,0,5,0,6,0
Input line #8:
3,0,2,0,8,0,1,0,0
Input line #9:
1,0,0,0,0,0,0,0,0
```

控制台输出答案
